# 粘性定位的使用

position:sticky：当元素在区域内时，元素不受定位的影响，当页面发生滚动时，定位会变成 fixed 定位，脱离文档流

    position: sticky;
    top: 10px;
1
2

# 粘性定位不生效的原因

position:sticky 的元素的效果完全受制于父级元素

  1. 父元素或者父辈元素设置了 overflow：visible的以外的任何值；
  2. 未设置 top、bottom、left、right 等属性
  3. 父元素的高度不能低于sticky元素的高度
